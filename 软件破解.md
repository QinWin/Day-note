### win10桌面图表箭头去除
1，新建shortcut.bat文件
```
@echo off

color 2

reg delete HKCR\lnkfile /v IsShortcut /f

reg delete HKCR\piffile /v IsShortcut /f

reg delete HKCR\InternetShortcut /v IsShortcut /f

taskkill /f /im explorer.exe && explorer
```
2，右键，管理员运行即可
